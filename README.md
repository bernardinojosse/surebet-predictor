# Algoritmo de Apuestas Seguras (Surebets) 🚀

Este proyecto utiliza web scraping para identificar oportunidades de arbitraje en diferentes casas de apuestas, garantizando un beneficio independientemente del resultado.

## 🧠 Idea del Algoritmo
1. **Web Scraping:** Obtención de momios en tiempo real.
2. **Normalización:** Emparejamiento de eventos entre distintas casas.
3. **Cálculo Matemático:** Aplicación de la condición de arbitraje.
4. **Optimización:** Cálculo de la apuesta ideal para maximizar ganancias.

## 📉 Condición Matemática
Para que una apuesta sea segura entre dos casas con cuotas $B$ y $A$, se debe cumplir:
$$(B - 1) \cdot (A - 1) > 1$$
